正方形视频

   <video id="myVideo" controls src="video.mp4"></video>

 

    <style>

        video {

            width: 300px;

            height: 300px;

            object-fit: fill;

        }

    </style>

 

    <script>

        // 获取视频元素

        var video = document.getElementById("myVideo");

 

        // 添加事件监听器

        video.addEventListener("ended", function() {

            // 在视频播放结束时，不执行任何操作

        });

    </script>

```

在这个示例中，`src` 属性设置为 `video.mp4` 表示视频文件为本地视频。在事件监听器中，并没有设置重新播放视频的操作，因此当视频播放结束时，不会重新播放。

 

正方形视频(有阴影)

  <div class="video-container">

        <video id="myVideo" controls src="video.mp4"></video>

    </div>

 

    <style>

        .video-container {

            display: inline-block;

            box-shadow: 0px 4px 8px rgba(0, 0, 0, 0.6); /* 增加阴影的大小和颜色强度 */

        }

 

        video {

            width: 300px;

            height: 300px;

            object-fit: fill;

        }

    </style>

 

    <script>

        // 获取视频元素

        var video = document.getElementById("myVideo");

 

        // 添加监听器

        video.addEventListener("ended", function() {

            // 视频播放结束时，不执行任何操作

        });

    </>

```

在述代码中，我们将 `box-shadow` 属性的值修改为 `0px 4px 8px rgba(0, 0, 0, 0.6)`， `4px` 表示阴影的垂直偏移量，`8px` 表示阴影的模糊半径，`rgba(0, 0, 0, 0.6)` 表示阴影的颜色和透明度。可以按需调整这些值来增加或减少阴影的明显程度。

 

长方形视频(无阴影)

    <video id="rectangleVideo" width="400" height="300" autoplay controls>

        <source src="video.mp4" type="video/mp4">

        Your browser does not support the video tag.

    </video>

```

请确保将名为 `video.mp4` 的视频文件放在与HTML文件相同的目录中，并将更新后的代码保存为HTML文件。在浏览器中打开该文件，您应该可以看到一个带有长方形视频和控制面板的页面。

 

长方形视频(有阴影)

   <style>

        .video-container {

            position: relative;

display: inline-block;

            box-shadow: 0 0 10px rgba(0, 0, 0, 0.5);

        }

 

        #rectangleVideo {

            position: relative;

            z-index: 1;

        }

 

        .video-container::after {

            content: "";

            position: absolute;

            top: -10px;

            bottom: -10px;

            left: -10px;

            right: -10px;

            z-index: 0;

            box-shadow: 0 0 20px rgba(0, 0, 0, 0.5);

        }

    </style>

</head>

<body>

    <div class="video-container">

        <video id="rectangleVideo" width="400" height="300" autoplay controls>

            <source src="video.mp4" type="video/mp4">

            Your browser does not support the video tag.

        </video>

    </div>

```

在上述代码中，我们将之前提供的CSS样式代码直接放置在 `<style>` 标签中，这样就不需要额外的CSS文件了。

 

确保将名为 `video.mp4` 的视频文件放在与HTML文件相同的目录中，并将更新后的代码保存为HTML文件。在浏览器中打开该文件，您应该会看到具有明显阴影的长方形视频播放器。并且CSS样式直接与HTML代码一起放置在一起。
